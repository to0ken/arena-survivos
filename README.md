# 🎮 ARENA - Survivors Arena






### Основные особенности:

| Особенность | Описание |
|-------------|----------|
| 🗺️ **Локации** | Лес, Пустыня, Завод — каждая требует свой ресурс (🍞/⛽/🔧) |
| ⚔️ **4 класса** | Воин, Лучник, Маг, Разбойник — уникальные стартовые оружия и бонусы |
| 🧙 **Волны врагов** | Динамическое усложнение с каждым раундом |
| 🎁 **Сундуки** | Выпадают с врагов, содержат ресурсы или предметы |
| 🔨 **Крафт** | Создание оружия и брони из материалов (дерево, железо, ткань) |
| 🛒 **Магазин** | Покупка предметов за внутриигровую валюту |
| 📦 **Инвентарь** | Система рюкзака с предметами и материалами |
| 🖼️ **Спрайты** | Асинхронная загрузка изображений из папки images/ |

---

## 🎮 Игровой процесс

### Схема игры:

```
Главное меню (выбор локации)
       ↓
Трата ресурса (провизия/топливо/инструменты)
       ↓
Загрузка арены (canvas 2400x1800)
       ↓
Выживание против волн врагов
       ↓
Сбор кристаллов опыта и сундуков
       ↓
Улучшение героя (навыки, оружие, броня)
       ↓
Переход на следующую волну (сложнее)
```

### Локации и ресурсы:

| Локация | Входной ресурс | Сложность |
|---------|----------------|-----------|
| 🌲 Лес | 1 🍞 Провизия | Средняя |
| 🏜️ Пустыня | 1 ⛽ Топливо | Высокая |
| 🏭 Завод | 1 🔧 Инструменты | Очень высокая |

### Экономическая система:

| Ресурс | Назначение | Способ получения |
|--------|-----------|------------------|
| 🍞 Провизия | Вход в локацию "Лес" | Сундуки, крафт |
| ⛽ Топливо | Вход в локацию "Пустыня" | Сундуки, крафт |
| 🔧 Инструменты | Вход в локацию "Завод" | Сундуки, крафт |
| 🌲 Дерево | Крафт оружия/брони | Сундуки |
| 🔩 Железо | Крафт оружия/брони | Сундуки |
| 🧵 Ткань | Крафт брони | Сундуки |

## 🚀 Запуск проекта

### Требования:
- Современный браузер (Chrome, Firefox, Safari, Edge)
- HTTP-сервер (из-за загрузки спрайтов)

### Инструкция по запуску:

**Шаг 1. Клонировать репозиторий:**
```bash
git clone https://github.com/ваш-username/arenaa-survivors.git
cd arenaa-survivors
```

**Шаг 2. Запустить локальный сервер (обязательно!):**

*Способ 1: Python*
```bash
python -m http.server 8000
# Открыть в браузере: http://localhost:8000
```

*Способ 2: VS Code Live Server*
- Установить расширение "Live Server"
- ПКМ на `index.html` → "Open with Live Server"

*Способ 3: Node.js*
```bash
npx serve .
```

**Шаг 3. Начать игру:**
- Выбрать героя (по умолчанию выбран Воин)
- Выбрать локацию (Лес/Пустыня/Завод)
- Нажать кнопку "Начать"
- Выживать против волн врагов!
